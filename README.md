# immersionmu

A starter Python application scaffold for the `immersionmu` repository.

## Running locally

1. Create a virtual environment:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```
2. Install the package in editable mode:
   ```bash
   pip install -e .
   ```
3. Run the app:
   ```bash
   python app.py
   ```

## Testing

```bash
python -m unittest discover tests
```
